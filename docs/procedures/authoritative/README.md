# Authoritative Procedures Index

Bu klasör, kritik governance, runtime ve security prosedürlerinin bireysel authoritative dosyalarını içerir.

## Kapsam
- policy exceptions
- approvals
- change and protocol governance
- release and hotfix flows
- incidents and postmortems
- preview and build triage
- rollback execution and validation
- tenant isolation, privacy and vendor security review

## Amaç
Paket prosedür belgelerini tek tek uygulanabilir dosyalara dönüştürmek.
