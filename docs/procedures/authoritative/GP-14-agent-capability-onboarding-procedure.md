# GP-14 — Agent Capability Onboarding Procedure

## Trigger
Yeni agent veya yeni capability sisteme ekleniyor.

## Steps
1. Capability declaration ve owner bilgisini kaydet.
2. Accepted inputs ve emitted outputs'u listele.
3. Policy ve protocol uyum kontrolü yap.
4. Fixture ve test refs ekle.
5. Rollout scope ve review sonucunu kaydet.

## Completion
- onboarding_completed
- onboarding_blocked
- onboarding_needs_changes
