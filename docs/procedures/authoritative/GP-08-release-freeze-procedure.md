# GP-08 — Release Freeze Procedure

## Trigger
Critical risk, incident veya governance hold.

## Steps
1. Freeze reason kaydet.
2. Etkilenen scope’u tanımla.
3. İlgili sahipleri bilgilendir.
4. Unfreeze şartlarını yaz.
5. Status ve audit kaydını güncelle.

## Completion
- freeze_active
- freeze_cancelled
