# RP-13 — Performance Degradation Response Procedure

## Trigger
SLO hedefi altında kalan performans bozulması gözleniyor.

## Steps
1. Affected service, route veya workflow scope'unu tanımla.
2. Metric refs ve baseline karşılaştırmasını topla.
3. Severity ve user impact özetini üret.
4. Mitigation veya rollback ihtiyacını değerlendir.
5. Follow-up owner ve deadline ata.

## Completion
- mitigated
- escalated
- monitoring_only
