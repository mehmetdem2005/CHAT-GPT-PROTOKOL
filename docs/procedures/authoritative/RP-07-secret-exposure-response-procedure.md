# RP-07 — Secret Exposure Response Procedure

## Trigger
Credential, token veya hassas config sızıntısı şüphesi oluştu.

## Steps
1. Etkilenen secret class'ı belirle.
2. Maruz kalan yüzeyleri listele.
3. Containment ve rotation ihtiyacını değerlendir.
4. Audit ve review kayıtlarını aç.
5. Doğrulama ve takip aksiyonlarını ata.

## Completion
- contained
- escalated_for_review
- false_positive
