# GP-11 — Postmortem Procedure

## Trigger
Yüksek etkili operasyonel olay veya tekrar eden ciddi problem.

## Steps
1. Timeline oluştur.
2. Root cause class yaz.
3. Contributing factors ekle.
4. Corrective actions ata.
5. Sahiplik ve tarihleri bağla.
