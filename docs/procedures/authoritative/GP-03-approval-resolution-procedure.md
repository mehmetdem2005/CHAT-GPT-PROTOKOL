# GP-03 — Approval Resolution Procedure

## Trigger
Açık approval request yanıtlandı.

## Steps
1. Approver kimliğini doğrula.
2. Self-approval yasağını kontrol et.
3. Kararı approved, rejected veya expired olarak kaydet.
4. Audit ve decision refs güncelle.
5. Turn state veya release gate etkisini uygula.

## Completion
- approved
- rejected
- expired
