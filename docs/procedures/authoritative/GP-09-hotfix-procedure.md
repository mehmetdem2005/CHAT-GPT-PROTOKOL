# GP-09 — Hotfix Procedure

## Trigger
Canlı veya kritik review yüzeyinde acil düzeltme gerekiyor.

## Steps
1. Hotfix kapsamını tanımla.
2. Normal akıştan sapma gerekçesini yaz.
3. Minimal change hedefini belirle.
4. Gerekli hızlı review zincirini çalıştır.
5. Sonradan tam review takibini aç.

## Completion
- hotfix_accepted
- hotfix_rejected
- hotfix_released
