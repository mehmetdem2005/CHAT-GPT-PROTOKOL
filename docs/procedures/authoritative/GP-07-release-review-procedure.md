# GP-07 — Release Review Procedure

## Trigger
Release candidate hazır durumda.

## Steps
1. Preview ve QA refs topla.
2. Policy ve risk özetini kontrol et.
3. Required reviewers listesini doğrula.
4. Sonuç kaydı oluştur.
5. Next action olarak continue, defer veya stop seç.

## Completion
- continue
- defer
- stop
