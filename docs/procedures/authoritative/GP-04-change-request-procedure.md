# GP-04 — Change Request Procedure

## Trigger
Schema, config, protocol, module boundary veya critical patch değişikliği.

## Steps
1. Change type belirle.
2. Impact summary yaz.
3. Target refs ekle.
4. Rollback veya mitigation yolunu ekle.
5. Review ve approval ihtiyacını işaretle.

## Completion
- accepted_for_review
- rejected
- needs_more_detail
