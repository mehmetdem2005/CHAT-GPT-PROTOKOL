# GP-12 — RCA Closure Procedure

## Trigger
Corrective action'lar tamamlandı veya kapatma değerlendirmesi gerekiyor.

## Steps
1. Action item durumlarını doğrula.
2. Kalıcı düzeltme evidence'ını ekle.
3. Riskin düştüğünü göster.
4. Problem kaydını kapat veya follow-up aç.

## Completion
- rca_closed
- followup_required
