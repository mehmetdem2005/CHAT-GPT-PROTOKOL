# GP-02 — Approval Request Procedure

## Trigger
Approval-bound change, riskli release veya major structural karar.

## Steps
1. Approval subject’ini tanımla.
2. Evidence refs topla.
3. Required approvers listesini çöz.
4. Blocking etkileri yaz.
5. Approval kaydını publish et.
