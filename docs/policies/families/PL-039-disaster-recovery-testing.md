# PL-039 — Disaster Recovery Testing Policy

## Objective
İş sürekliliği ve recovery planlarının yalnız belgede kalmamasını, test edilmesini sağlamak.

## Scope
- DR drills
- restore rehearsals
- failover validation
- recovery evidence

## Mandatory Checks
1. Recovery path düzenli tatbikat veya rehearsal taşımalı.
2. Başarısız tatbikat follow-up action üretmeli.
3. DR readiness yalnız varsayım değil evidence ile desteklenmeli.

## Enforcement
- require_review
- release_hold

## Evidence
- drill refs
- restore validation refs
- follow-up actions
- readiness notes

## Review Cadence
Quarterly veya risk seviyesi yüksek yüzeylerde daha sık.
