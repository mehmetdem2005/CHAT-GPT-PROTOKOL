# PL-031 — Compliance Certification Policies

## Objective
Uyum kanıtları, kapsam eşleme ve breach notification readiness beklentilerini korumak.

## Primary Subjects
- compliance evidence
- releases
- policy changes

## Mandatory Checks
1. Scope mapping ve evidence refs görünür olmalı.
2. Breach ve notification davranışı belgelendirilmeli.
3. Uyum boşlukları release kararına bağlanabilmeli.

## Enforcement
- require_review
- require_approval
- hard_block

## Exception Path
Yalnız documented compliance sign-off ile sınırlı istisna mümkündür.

## Evidence
- control evidence
- scope maps
- release refs
- notification procedures

## Review Cadence
Quarterly ve certification scope değiştiğinde.
