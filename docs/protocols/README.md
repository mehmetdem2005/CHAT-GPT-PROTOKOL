# Protocol Documentation Index

Bu klasör, sistemin önceki protokol omurgasını unutmayacak şekilde resmi protokol belgelerini içerir.

## Amaç
- Ajanlar arası ortak iletişim omurgasını repo içinde kalıcı hale getirmek
- Ayrı sohbetlerde geliştirilen modüllerin aynı protocol-first yaklaşımına bağlanmasını sağlamak
- Önceki çekirdek protokol belgelerini kaybetmeden yeni mimariyle birleştirmek

## Dosya Yapısı
- `legacy-protocol-lineage.md` — İlk protokol omurgasının soy ağacı ve kapsamı
- `master-protocol-document.md` — Contract-first sistemin ana anayasa belgesi
- `protocol-registry-v1.md` — Çekirdek protokol kayıt defteri
- `turn-state-schema-v1.md` — Çok turlu çalışma ve konuşma state modeli
- `artifact-manifest-schema-v1.md` — Artifact ownership ve traceability şeması
- `preview-runtime-protocol-spec-v1.md` — Build, runtime ve preview çalışma protokolü
- `intervention-and-rollback-spec-v1.md` — Müdahale ve geri dönüş spesifikasyonu

## Kapsam
Bu klasörde özellikle şu erken omurga korunur:
- Master Protocol Document
- Protocol Registry v1
- Turn State Schema v1
- Artifact Manifest Schema v1
- Preview Runtime Protocol Spec v1
- Intervention & Rollback Spec v1

## Kullanım Notu
Bu protokoller, daha sonra eklenen policy, observability, auth, deployment ve governance belgeleriyle birlikte okunmalıdır.
