# Next-Wave Remediation Plan v1

Bu belge, dürüst değerlendirmede işaretlenen ikinci dalga eksiklerin kapatılması için resmi iyileştirme planını tanımlar.

## Amaç
- kısa kalan policy family dosyalarını derinleştirmek
- prosedürleri operasyonel hale getirmek
- M1 protokolleri schema-ready seviyeye yaklaştırmak
- duplicate ve drift problemlerini azaltmak
- eksik standards ve registries katmanını kapatmak

## Dalga 2 Öncelikleri
1. Policy family deepening
2. Procedure operationalization
3. Protocol schema deepening
4. Canonicalization and deprecation cleanup
5. Documentation entry and audit refresh

## İlk Uygulama Hedefleri
- PL-001, PL-006, PL-007 için extended family spec
- GP-01, RP-07, RP-11, RP-13 için extended procedure spec
- API Style Guide, Observability Standard, Testing Standard
- Accessibility Audit Procedure
- Auth Incident Response Procedure
- Data Deletion and Correction Procedure
- Release procedure canonicalization
- Coverage gap audit v2
- Root README / entrypoint doc

## Kural
Bu dalga da non-breaking in-place upgrade yaklaşımıyla yürütülür.
